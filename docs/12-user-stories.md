User Story 1

As a discharge coordinator,

I want CareGuardian to identify missing discharge information,

so that patients are not discharged with incomplete records.

User Story 2

As a discharge coordinator,

I want CareGuardian to identify pending test results,

so that no important investigations are forgotten.

User Story 3

As a nurse,

I want to receive alerts when follow-up ownership is unclear,

so that patient care responsibilities are properly assigned.

User Story 4

As a patient safety officer,

I want to review trends in discharge-related risks,

so that preventable harm can be reduced.
